# **App Name**: SQL Accelerator

## Core Features:

- Interactive Course Overview: A visually engaging and interactive overview of the SQL course curriculum.
- Hands-on Project Showcase: Showcase the real-world mini-project, highlighting its key features such as real-world datasets and query optimization, which act as a tool for solidifying learning.
- Contact Form with Email Integration: A contact form that integrates with Resend via Supabase Edge Functions to send query confirmations upon submission. Requires Resend API key configuration.
- Responsive Single-Page Layout: A fully responsive single-page layout optimized for mobile, tablet, and desktop devices with touch-friendly components.

## Style Guidelines:

- Primary color: Dark blue (#003366) to convey professionalism and trust.
- Background color: Light gray (#f0f0f0) to provide a clean and neutral backdrop.
- Accent color: A slightly brighter blue (#4682B4) to draw attention to important interactive elements like buttons and links.
- Headline font: 'Poppins' (sans-serif) for a modern and clean look.
- Body font: 'PT Sans' (sans-serif) offering great readability.
- Clean and professional icons representing SQL topics and course benefits.
- Subtle fade-in effects, hover animations, and smooth scrolling to enhance user experience.
